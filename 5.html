Do you + 동사 ? : 너는 + ~ 하니?/~해?

  • Do you
  • wash? / wash you car?
  • make? / make tea?
  • eat? / eat lunch?
  • drink? / drink water?
  • clean? / clean the bedroom?
  • read? / read the newpaper?
  • write? / write a story?
  • study? / study Chinese?
  • watch? / watch the game?
  • play? / play the piano?
  • 너는 씻니? / 너는 세차하니?
  • 너는 만드니? / 너는 차를 만드니?
  • 너는 먹니? / 너는 점심을 먹니?
  • 너는 마시니? / 너는 물을 마시니?
  • 너는 청소하니? / 너는 침실을 청소하니?
  • 너는 읽니? / 너는 신문을 읽니?
  • 너는 쓰니? / 너는 이야기를 쓰니?
  • 너는 공부하니? / 너는 중국어를 공부하니?
  • 너는 지켜보니? / 너는 경기을 보니?
  • 너는 연주하니? / 너는 피아노를 연주하니?

car 자동차. tea (마시는)차. lunch 점심. water 물. bedroom 침실. newpaper 신문. story 이야기. Chinese 중국어. game 게임/경기. piano 피아
